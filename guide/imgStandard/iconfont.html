    iconfont

    考虑到移动端图片太多影响加载效率，因此本团队，对移动端使用icon字体库（PC端按情况而定）

    • 优点

      • 减少请求次数，将多个 icon 合并到一个字体文件中，从而提高网页性能；
      • 自由的变化大小和颜色，因为字体是可以通过样式来设置大小和颜色的；
      • 矢量图不失真；
      • 适合多平台（浏览器、iOS App、Android App）；

    • 缺点

      • 单色
      • 维护成本较高

    iconfont 引入

    本团队没有自己的iconfont库，因此用的是第三方iconfont库 (opens new window)

    注意

    相关注册流程请联系相关人员帮助注册使用

    实例

    /* 在main.scss文件中引入相关代码 */
@font-face {
  font-family: 'iconfont';  /* project id 1397066 */
  src: url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.eot');
  src: url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.eot?#iefix') format('embedded-opentype'),
  url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.woff2') format('woff2'),
  url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.woff') format('woff'),
  url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.ttf') format('truetype'),
  url('//at.alicdn.com/t/font_1397066_d86rcowj3fh.svg#iconfont') format('svg');
}
.iconfont {
  font-family: "iconfont" !important;
  font-size: 0.8rem;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -webkit-text-stroke-width: 0.2px;
  -moz-osx-font-smoothing: grayscale;
}
